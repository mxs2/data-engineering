# Entrega

A atividade deverá ser realizada em grupo de até 3 pessoas.

Na entrega, é obrigatório informar o nome de todos os integrantes do
grupo.

# Requisitos

O projeto deve seguir as boas práticas de código trabalhadas em aula,
incluindo:
- inclusão de docstrings nos métodos desenvolvidos;
- uso da biblioteca Black para padronização e formatação do código;
- criação de um arquivo README com as informações do projeto.
